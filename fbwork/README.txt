Required Libraries:
1. pandas ( to install: pip install pandas)
2. numpy ( to install: pip install numpy)
3. sklearn ( to install: pip install sklearn)


Steps to run:
1. Run Divide_data.py
2. Run predictive_model.py (if you want to train classifier again else skip the step)
3. Run test.py
4.Extended_test.py

To test the classifier of different data set
1. Create csv file of data with same columns.
2. open Extended_test.py replace the "testdata.csv" with the name of csv file you have created.
